Prototyping - Figma

I believe that Figma is an excellent tool for prototyping. No need for complicated and expensive apps, no complicated interface. But, the most important element is work in browser environment. Figma isn't tool only for prototyping. Wireframing is in modern design very important step in creation of quality UX/UI design layouts and Figma provide the set of tools, which can help to you to create quality wireframe elements.
