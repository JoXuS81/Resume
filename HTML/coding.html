Web Development Tools in 2020

I work with several new web design tools in 2020. I added to my collection Figma and Visual Studio Code editor. Figma is high class online design tool, which is very similar to Photoshop or Illustrator. It's an excellent tool for work on prototypes and wireframes in web development. On the other hand, I am very satisfied with Visual Studio Code. I heavy use SCSS, and need to have a good SCSS compiler. Because of very good extensions library support, Visual Studio Code is excellent for me, because it have an extension called - SASS Live Compiler.
